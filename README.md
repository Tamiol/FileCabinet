# FileCabinet
***

#### Local Setup

1. Clone the repository using Git:
    ````
    git clone https://github.com/Tamiol/FileCabinet.git
    ````
2. Go into the project directory:
    ````
    cd FileCabinet
    ````
3. Build and run application:
   ````
   mvn clean install
   mvn exec:java
   ````
